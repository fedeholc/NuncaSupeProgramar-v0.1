Para crear un nuevo proyecto:

```sh
npx create-next-app@latest
```

Para cambiar el puerto en el que se ejecuta, cambiar en el `package.json`:

```json
"scripts": {
  "dev": "next -p 3002",
  "build": "next build",
  "start": "next start -p 3002",
},
```
